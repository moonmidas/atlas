# 03 - Data Products

**Category:** Software  
**Project Range:** $10K-$75K+  
**Timeline:** 3-6 months to MVP, 12-18 months to $30K MRR  
**Business Model:** Subscription ($100-$1,000/month) + Enterprise ($10K-$100K/year)

---

## 🎯 The Opportunity

Data Products = Intelligence subscriptions built on operational, market, or competitor data. You aggregate data that's expensive to gather manually, add analysis/insights, and sell access. High margin (90%+), defensible (data moat), scalable.

### Why It Works
- Data collection is expensive and tedious
- Decisions are made with weak or no data
- Information asymmetry = profit opportunity
- API and dashboard access = sticky subscriptions

---

## 💰 Economics

| Metric | Range |
|--------|-------|
| Entry Investment | $10K-$35K (data sources + scrapers + MVP) |
| Monthly Burn | $3K-$10K (founder + data costs) |
| Target MRR | $20K-$100K |
| Target Pricing | $200-$2,000/month |
| Gross Margin | 85-95% |
| Data Refresh | Real-time, daily, or weekly |

**Exit Multiples:** 4-8x ARR (lower than SaaS but higher margins)

## 🎯 Positioning Strategy

### The Hook
"Stop paying [expensive tool] $X,000/year for data that's 6 months old. Get real-time [industry] intelligence for $Y/month."

### Validation Path
1. **Week 1-2:** Identify data source and validate demand (20 interviews)
2. **Week 3-4:** Build MVP scraper, collect initial dataset
3. **Month 2:** Launch beta to 10 customers, iterate
4. **Month 3-6:** Scale data collection, add features
5. **Month 6-12:** Scale to $20K MRR

### Pricing Strategy
- **Starter:** $200/month (dashboard only)
- **Pro:** $500/month (API access, exports)
- **Enterprise:** $2,000+/month (custom data, support)

---

## 🚨 Red Flags

- ❌ Data that changes too slowly (annual refresh only)
- ❌ Data you can't legally access or scrape
- ❌ Markets where incumbent has exclusive data agreements
- ❌ Data that's easy for customers to collect themselves
- ❌ Opportunities requiring $100K+ to build dataset

---

## 💡 Success Patterns

**The Formula:**
1. Find data that's expensive to gather manually
2. Build scrapers/collection infrastructure
3. Launch with 90-day free trial to build dataset
4. Price at 20% of manual collection cost
5. Add proprietary analysis on top of raw data

**Examples to Study:**
- CB Insights - Startup/VC data - $100M+ revenue
- Crunchbase - Company data - $50M+ revenue
- Second Measure - Transaction data - $100M+ acquisition
- Thinknum - Alternative data - $50M+ funding
